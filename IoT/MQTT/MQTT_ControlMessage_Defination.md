# MQTT 控制报文格式



## MQTT 控制报文的结构

MQTT 协议通过交换预定义的MQTT控制报文来通信。

MQTT 控制报文由三部分组成，按下表描述的顺序：

| Fixed header 固定报头，所有控制报文都包含  |
| :----------------------------------------: |
| Variable header 可变报头，部分控制报文包含 |
|     Payload 有效载荷，部分控制报文包含     |



## 固定报头

每个MQTT控制报文都包含一个固定报头。下表描述了固定报头的格式：

|  Bit   |  7   |  6   |  5   |  4   |  3   |  2   |  1   |  0   |
| :----: | :--: | :--: | :--: | :--: | :--: | :--: | :--: | :--: |
| byte 1 |      |      |      |      |      |      |      |      |
| byte 2 |      |      |      |      |      |      |      |      |
